# EasyCommute

EasyCommute is a full-stack web application designed to simplify daily commuting through efficient route handling, real-time communication, map-based visualization, and integrated online payments. The project was developed as part of a Third Year academic curriculum, focusing on building a practical, production-ready system using modern web technologies.

## Live Demo

https://easy-commute-one.vercel.app/

## Overview

The application enables users to manage and optimize their daily commute with a responsive interface and real-time updates. It uses WebSocket-based communication for dynamic interaction, integrates map visualization using Leaflet, and includes secure payment processing.

## Features

- Real-time communication using Socket.IO  
- Interactive maps powered by Leaflet  
- Secure payment integration via Razorpay  
- Responsive, mobile-first user interface  
- Modular and scalable architecture  
- Efficient client-server interaction  

## Tech Stack

### Frontend
- React  
- Tailwind CSS  
- Vite  
- Leaflet  

### Backend
- Node.js  
- Express.js  
- Socket.IO  

### Integrations
- Razorpay Payment Gateway

## Usage

- Access the application via the live URL or local setup  
- Visualize routes and locations on interactive maps  
- Real-time GPS tracking  
- Complete transactions using Razorpay
